## プロモーションテキスト (170字)

## 説明文 (4000字)
気になったこと、やりたいこと。SNSやWebサイトで見つけて「いつかやろう」と思ったことを、そのまま放置していませんか？

Wishes は、その「いつか」を、ちゃんと「やった」に変えるためのアプリです。ただ管理するだけでなく、“叶える”ところまで、そっと手伝います。

なんとなく過ごしてしまっていた日々を、少しずつアップデートしませんか？

■ 他のアプリからでも、ワンタップで保存。
Safari やマップ、SNS で気になったお店・場所・記事は、共有ボタンから Wishes へ。アプリを切り替えず、思いついた瞬間にそのまま追加できます。

■ 面倒な入力は、AIにおまかせ。
URL を渡すだけで、AI がタイトル・シーズン・タグまで自動で下書き。ぽちぽち手入力する手間は、もういりません。（AIの処理はすべて端末内で動作します）

■ ゆるく整理。タグ地獄にはしない。
優先度（高・中・低）／シーズン（春夏秋冬・月・いつでも）／タグ（飲食・旅行・レジャー・お買い物＋自由に追加）の3軸で、ざっくり分類。仕事のタスク管理みたいに、しんどくなりません。

■ ちょうどいいタイミングで、そっと提案。
金曜は今週末、月初は今月、季節の前にはその季節——。今のあなたに合う「いつか」を、最大3つだけ、そっと差し出します。

■ 叶えた数が、増えていく。
チェックを付けると達成として記録。今年いくつ叶えたかがひと目でわかり、あなたの希望ペースに合わせて、次の一歩をそっと後押しします。

■ プライバシーは最優先。
データはすべて端末内に保存。アカウント登録は不要で、広告・トラッキング・分析は一切ありません。

────
やりたいことが、ないわけじゃない。ただ、埋もれているだけ。
Wishes で、あなたの「いつか」を、ひとつずつ「やった」に。

## App Review に関する情報（審査メモ）
審査担当者向けに以下を記入（英語推奨だが日本語審査なら日本語可）:

- 本アプリはアカウント不要・ログイン不要です（デモ用アカウントは必要ありません）。
- データはすべて端末内（App Group）に保存し、外部サーバーへ送信しません。
- 「AI」機能は Apple のオンデバイス基盤モデル（対応端末）またはルールベースで動作し、外部APIを呼びません。共有・貼り付けられたリンクは、タイトル/タグの下書きのために端末上でそのリンク先のみ取得します。
- 本バージョンは課金機能なしの無料アプリです（アプリ内課金は提供していません）。URL自動取り込みを含む全機能を無料で利用できます。
- 動作確認手順: ホームの「＋」から追加シートを開き、タイトルを入力して保存 →リストに追加されます。URL欄にリンクを貼ると端末内で下書きが生成されます。共有シート（Safari等の共有→Wishes）からも追加できます。
