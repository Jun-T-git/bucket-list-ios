## プロモーションテキスト (170字)
SNSで見つけた「いつかやりたい」、保存して終わっていませんか？バケットリストは、あなたの“やりたい”を適切なタイミングでそっと差し出して、ちゃんと消化させるためのアプリです。

## 説明文 (4000字)
「なんとなく終わった」を減らすためのアプリです。

やりたいことはある。SNSに保存もしている。でも、気づけば時間が過ぎている——。
本質的な問題は「やりたいことが無い」ことではなく、「やりたいことが埋もれる」こと。バケットリストは、未来の自分に書いた“いつか”を、適切なタイミングで差し出します。

これは管理アプリではありません。消化を手助けするアプリです。

■ ためる（回収）
思いついたこと、SNSやニュースで見つけたものを、共有ボタンからワンタップで登録。リンクを渡すと、端末の中だけでタイトルやタグを下書きします。

■ ゆるく整理（3軸）
・優先度（高 / 中 / 低）
・シーズン（春夏秋冬・月・いつでも）
・タグ（飲食 / 旅行 / レジャー / お買い物 ＋ 自由に追加）
タグ地獄にはしません。シンプルさを優先しています。

■ ちょうどいいタイミングで提案
金曜は今週末、月初は今月、季節の前にはその季節——。今のあなたに合うものを、最大3つだけそっと提案します。

■ 達成を記録してふり返る
チェックを付けると達成として記録。今年いくつ叶えたかが見えるので、「やった」が積み上がっていきます。

■ プライバシー最優先・オンデバイス
データはすべて端末内に保存。アカウント登録は不要で、広告・トラッキング・解析はありません。AIによる下書きも端末内で動作し、あなたの入力を外部に送信しません。

————
仕事のタスク管理ではなく、人生をちょっとよくするための小さな相棒として。
あなたの「いつか」を、ちゃんと「やった」に変えていきましょう。

## App Review に関する情報（審査メモ）
審査担当者向けに以下を記入（英語推奨だが日本語審査なら日本語可）:

- 本アプリはアカウント不要・ログイン不要です（デモ用アカウントは必要ありません）。
- データはすべて端末内（App Group）に保存し、外部サーバーへ送信しません。
- 「AI」機能は Apple のオンデバイス基盤モデル（対応端末）またはルールベースで動作し、外部APIを呼びません。共有・貼り付けられたリンクは、タイトル/タグの下書きのために端末上でそのリンク先のみ取得します。
- 本バージョンは課金機能なしの無料アプリです（アプリ内課金は提供していません）。URL自動取り込みを含む全機能を無料で利用できます。
- 動作確認手順: ホームの「＋」から追加シートを開き、タイトルを入力して保存 →リストに追加されます。URL欄にリンクを貼ると端末内で下書きが生成されます。共有シート（Safari等の共有→バケットリスト）からも追加できます。
