# 0005. v1.0 は無料。Pro は実装済みだがフラグで無効

- Status: Accepted
- Date: 2026-06〜07

## Context
マネタイズは「買い切り Pro」（URL自動取り込み/オンデバイスAI補完を無料枠10回超で Pro 化）を想定し、
StoreKit2 で実装済み。だが v1.0 提出時点では、まず無料で世に出して母数と反応を得たい。
また Paid Apps 契約や ASC 上の IAP 作成は提出の手間を増やす。

## Decision
- v1.0 は**課金機能を無効化して無料リリース**する：`FeatureFlags.proEnabled = false`。
- StoreKit2 コード（`ProStore` / `PaywallView`）と無料枠（`Storage.freeCaptureLimit = 10`）は**残す**が、
  `proEnabled == false` の間、`canAutoCapture` は常に true・`consumeFreeCapture()` は no-op。
- App Store 提出では Paid Apps 契約・IAP 作成をスキップ（[release-runbook](../workflows/release-runbook.md)）。

## Consequences
- 全ユーザーが URL 自動取り込みを無制限に使える。ストア文言も「全機能無料」で統一。
- 将来 Pro を有効化する際は `proEnabled` を true にし、ASC で IAP（`teratech.BucketList.pro`）を作成
  ＋ Paid Apps 契約（手順とメタデータは [app-store-metadata](../workflows/app-store-metadata.md) のIAP節=将来参照用）。
- 拡張は StoreKit を動かせないため、エンタイトルメントは App Group suite にミラー（`Storage.proEntitled`）。
